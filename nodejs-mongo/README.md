NodeJS & MongoDB
=================

Running MongoDB queries using node.js
  
### Usage 
1. Complete connection string
2. Complete collection name
3. execute run.js
```
$> node run.js
```


###Sample output
````
$>node run.js
inserting .....0
inserting .....1
inserting .....2
[ 'connect success', 'insert success' ]
close db
$>
````
